# Pensieve

An Obsidian plugin for AI-enhanced note management — optimized for better AI consumption of your vault.

## Installation

### Manual

1. Download the latest release from the [Releases](https://github.com/sebastienlevert/obsidian-pensieve/releases) page
2. Extract into your vault's `.obsidian/plugins/pensieve/` directory
3. Enable the plugin in Obsidian settings

## Development

```bash
npm install
npm run dev    # Watch mode
npm run build  # Production build
```
